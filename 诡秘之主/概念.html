愚者 #

The Fool

愚者是塔罗牌中的一张牌，通常是一张未编号的牌，代表着新的开始、冒险和冒险家的形象。这张牌通常描绘了一个年轻的男性，他背着一个小包袱，手持一根棍子，正在一条狭窄而陡峭的小路上前行。他的姿态轻松自如，似乎对前方的未知世界充满了好奇心和兴奋感。他的服装和装饰也显示出他的天真和单纯，例如他可能穿着明亮的衣服或带有鲜艳图案的帽子。在他身边可能有一只小狗，也是象征着忠诚和信任的动物。

愚者并没有编号（或作0号又或作22号），是所有塔罗牌的开始，亦代表着塔罗牌的结束，代表着无限的可能性，以及混乱、纯真、疯狂的原始能量。

魔术师 #

I-The Magician

伟特塔罗牌图像中，一名穿着红色披肩的男子站在摆有权杖、圣杯、宝剑、钱币的桌子后方，一手指向天、一手指向地，头顶有着象征无限大的符号，身边环绕红色玫瑰与白色百合。

正位牌意有擅于社交、资源运用得当、思考敏捷、具有良好的专业技巧，同时也具有开展，开端，以及准备充分的意义；逆位牌意则象征手法粗造、欺骗、才华被埋没，有时也提示为不适合发展新的方向，或准备不充足。敏捷的思考、注重交流的意义特色，在占星术中与代表思考与交通的水星相对，有时也可象征兄弟手足与同侪。

女祭司 #

II-The High Priestess

在韦特牌中，女祭司被描绘成舍吉拿的形象。她穿着朴素的长袍，手放在大腿上端坐。她的脚上有一个新月，类似于古埃及女神哈索尔，胸前有一个十字架。她手上的卷轴上写着字母TORA。她坐在黑白的柱子之间，字母J和B象征着Jachin and Boaz，所罗门圣殿的两根柱子；神庙的帷幔在她后面。

皇后 #

III-The Empress

在韦特的定义中，她不是天之女皇，而是罪人的避难所：一个多产的母亲，她是高于一切的繁殖力，培养和维护世界的一切。

皇帝 #

IV-The Emperor

以自我为中心开始做事。皇帝是强而有侵略性的代表。手执白羊座的权杖(火的象征)和球(地的象征)。在背景中的两头羊亦代示这张牌很代表到白羊座的两面：左面深红色的羊代表白羊座暴力和有害的能量；右边金黄色的羊代表白羊座的能量亦可以有建设性和有利的一面。皇帝手执权仗便代表他能决定和行使这股力量。

国家这个意识亦在牌面上表达出来。皇帝的胸前盔甲上的数只蜜蜂象征了以个人的努力去成全整体。亦表示皇帝是国家的象征。要在两只羊中取得平衡而非在图中左下的两只鸟惹起斗争，在必要的时候亦要像右下的羊以圣名挑起战旗。

教皇 #

V-The Hierophant

它代表着神圣的秘仪和灵性的权威，以及在宗教、宗教仪式和传统中的力量和地位。在牌面上，这位被描绘为身穿长袍、戴着三重冠的男子，他坐在一个装饰华丽的宝座上，手持着一把执杖，另一只手则举起一个手势，似乎在发表正式的声明或宣布祝福。

这张牌通常被视为是精神领袖、导师、守护者、传统领袖和仪式主司之类的代表。它提醒我们要保持谦虚和敬畏，对生命和更高次元的意义保持敏感，并意识到我们的确是无足轻重的个体，只有通过信仰和众生的支持才能在人生之路上前行。

恋人 #

VI-The Lovers

恋人通常被描述为一个男人和一个女人在一座园林中一起站着，他们背后有一朵罗勒花树，上面有一条毒蛇。女人向男人伸出手，手中握着一个苹果，男人手中握着火炬，似乎在宣誓永恒的忠诚。这张牌代表着选择、爱情、伴侣关系、性、平衡和协调。

在塔罗牌阅读中，这张牌经常涉及到个人的选择和抉择，它提醒人们要做出有意识、负责任的个人选择，并考虑到所有相关的可能性和影响。它也可以代表一个平衡的、有爱的伴侣关系，需要互相支持和理解，同时保持独立和自我。在某些情况下，这张牌也可能意味着需要做出具有挑战性的、具有决定性意义的选择，并准备好为此付出相应的代价。

战车 #

VII-The Chariot

战车的图像是一个王子样子的人物驾驭一辆战车，通常由一黑一白的两个斯芬克斯或两匹马拉动，人物戴王冠或头盔，持剑或魔杖等物。

战车代表胜利、征服，或者势力、冲突[2]。与占星学上的巨蟹座相对应[3]。

力量 #

VIII-Strength

力量是塔罗牌中的第8张牌，表示心理、情感或精神上的力量和稳定。这张牌通常描绘为一个女人手握一头狮子的嘴巴来表达她对这种力量的掌控。女人通常被穿着白色衣服，代表她的纯洁和意志坚定。 狮子则表示原始的动力和冲动，但是因被女人的控制而得到了平衡。该牌象征者通过最耀眼的力量和柔软和谐之间的平衡来表示潜在的胜利。

这张卡牌的倒立，表示一种力量的缺失或者失去了掌控的力量，可能导致情绪失控或决策错误。

隐者 #

IX-The Hermit

隐士是塔罗牌中的第9张牌，通常代表着孤独、内省和寻求真相的旅程。这张牌通常描绘为一个老人单独站在山顶上，手持一只油灯，代表着他关注内在的事物，而不是外部世界。他的长袍和拐杖也显示出他的智慧和故事的经验。他代表了一个人面对自己的内心和灵魂，独自一人在寂静中寻求启示。

隐士的倒立表示一个人的内在旅程遇到障碍或挫折。它可能意味着缺乏内省或寻求真相的意愿，或者走向了错误的方向。这张牌强调需要内省和寻求真相以避免迷失方向。

命运之轮 #

X-Wheel of Fortune

与I 魔术师及O 愚者相对，都是意味着有新的开始。亦与占星术中的木星相对，木星占星术中是一颗吉星，所以命运之轮一般而言是带来幸运的一张牌。

牌面中心的圆轮象征命运之轮。命运之轮旁左侧的蛇象征邪恶，蛇头向下代表命运的沉沦；右侧背负命运之轮者为埃及神明阿努比斯，代表背负命运上升的渴望；命运之轮正上方盘坐者为人面狮身，代表智慧、抉择与平衡。牌面四角的动物分别是象征宝瓶座的人、狮子座的狮子、金牛座的牛及天蝎座的鹰，代表四大元素，即宝瓶座—风、狮子座—火、金牛座—土、天蝎座—水。

正义 #

XI-Justice

正义是塔罗牌中的一张牌，通常描绘为一位坐在宝座上的女性，她手持天秤和剑。这张牌代表公正和平衡，涵盖法律和道德方面的概念。它传达了一种“公正”主题，意味着在一个决策中要先考虑规律、原则和平衡。

倒吊人 #

XII-The Hanged Man

通常画面上是一个男性，头向下垂，右腿跪在左脚上，左手背向外部，右手被绑在身后的树枝上。

塔罗牌作者伊娃盖伦在她的著作中说这张牌代表要做出艰难而痛苦的牺牲，但这个牺牲将带来更好的结果。这也反映出神学上的基督教象征，即为救赎人类而牺牲的复活节、圣经以及圣经中的最后晚餐。

有时，绞死者也表示放下欲望和放弃过去的事物，让自己可以有新的开始。牌的确切意义依赖于阅读者和牌组的意图，以及塔罗牌牌面中其他牌的影响。

死神 #

XIII-Death

这张牌通常描绘一个人骑在马上，手里拿着旗帜，旁边有一个正在死去的人、一个哀悼的人和一个被割断的头。这些象征着生命的终结和新的开始。尽管这张牌被称为死亡，但它实际上代表着抛弃旧的、不必要的东西，以便为更多的机会和成长让路。

节制 #

XIV-Temperance

代表平衡、和谐和自律。这张牌通常描绘一个天使站在两个杯子之间，从一杯子中倒出液体到另一个杯子中，要求天使塑造平衡、谨慎和适度。这张牌表明了需要在生活中取得折衷，同时保持心情平稳和集中，以及在行动中秉持自律并保持克制。它强调了内部和外部因素之间的平衡，以创造和谐和良好的关系。

恶魔 #

XV-The Devil

通常表示恶魔、妖术、诱惑和不道德。在这张牌中，出现一个被深色背景覆盖的人形物体，长角、羊头、手里拿着一个火炬，在它的脚下，有一个男人和一个女人被链子紧紧地束缚着。它代表着一种亵渎的力量，表明受到生理和心理的束缚，缺乏自由和控制。 该牌提醒人们注意可能会损害精神和健康的行为和想法，并建议寻求变革自己的行为，以获得自由和更好的机会。

塔 #

XVI-The Tower

通常代表着改变、毁灭和重建。这张牌通常描绘一座高塔被闪电击中，塔上的人被震惊，并从塔上跌落，暗示灾难的出现。 这张牌揭示了某种力量猛烈地摧毁了现状，在这个过程中旧的观念和信念体系被根本颠覆，但也提醒人们，重建、重组和重申是有可能的，这为新的生活带来了希望和机遇。这张牌鼓励我们要乐观面对，尽管存在风险和阻力，但也要坚定地寻找新的出路。

星星 #

XVII-The Star

通常代表信仰、希望和冥想。这张牌通常描绘一个裸体的女性，站在一条小溪旁边，她的手举在头上，握着一颗大星星，周围还有其它小星星。这张牌象征着内在和平、神圣的力量和灵感，它代表了透过信仰、宣扬希望和深入冥想，你能获得心灵的启示和洞见。 它提醒人们，在黑暗的时刻也要抱持信念，因为星星的出现代表了新的开始、平静和平衡的到来。这张牌强调对艺术、诗歌和其他人类文化的重视，并因此鼓励个人去适应它们。

月亮 #

XVIII-The Moon

通常代表变化、神秘和双重性。这张牌通常描绘一个躲在两个巨大的石柱后的狗和狼，月亮和星星在上面，一条小路向远方延伸。这张牌表明，人们面对的现实并不一定是他们所看到的，而是隐藏在其间的神秘力量所掌控着。它强调灵性现象与物质现实之间的连接，提醒人们注意潜意识和直觉的影响。这张牌意味着变革、无所预料和挑战，因此，它要求人们要更谨慎地处理问题，并更加关注梦、幻象和直觉。这张牌也可能是一个警告，因为它代表着欺骗、误导和幻觉。

太阳 #

XIX-The Sun

通常代表明亮的前景、成长和成功。这张牌通常描绘一个小男孩骑在马上，太阳照耀在他们身上，预示着欣欣向荣的远景。牌面图案代表着新的开始，新的起点和机遇，同时它还象征着创造力、清晰思考和内心与外部世界之间的协调。这张牌意味着天真、快乐和成功，代表着情感和个性上的完全开放，身心的生气和重生。它具有强大的治愈力和疗愈力，并能够带来幸福和良好的未来。工作上，该牌象征着需秉持努力、自信和新的起步心态，并带着希望和乐观的心态去寻找成功。

审判 #

XX-Judgement

通常代表重生、清算、自我反省和审判。这张牌通常描绘一个天使在吹响一根号角，从坟墓中升起来的人们向他走去，表示重生和自我反省的开始。它代表了集中力量、审视自己和自我反省的过程，以准备未来的改变和转型。这张牌还强调了个人自我审判和自我重生的重要性，以及面对挑战和困难时如何接受过去的错误和不幸。审判牌还提示人们，必须秉承公正、真理和誓言的原则，认真承担自己的行为和选择，并为和谐、公正和平等而奋斗。

世界 #

XXI-The World

通常代表完成、成就和宇宙的和谐。这张牌通常描绘一个裸体的女性站在一个大圆圈之中，四个象征着四个元素的形象在圆圈的四个角出现。这个女人代表了完美的完成和宇宙和谐的象征。 它表示对自我和周围环境的充分认识，并且能够以全球性视角看待事物。它还象征着个人的成功、充实和满足，以及对过去所做努力取得的成果的庆祝和肯定。这张牌提醒人们，必须以开放、平衡和清醒的眼光，如此才能在未来获得成功，同时也暗示着继续实现个人和宇宙和谐的进程。

神秘学 #

占卜 #

很多时候，占卜来源于自身，每个人都有灵性，都能交感到灵界，交感到更高层次的关系自身的信息，只是普通人无法察觉这点，更加别说去解读获得的提示了，当他们使用占卜工具的时候，这些信息就会借助工具呈现出来，一个最简单的例子，梦和解梦。塔罗牌实际上就属于这种工具，它用更多的象征，更合理的元素，帮助我们更方便更准确地解读提示。

转运仪式 #

灰雾之上 #

无垠的灰雾如水流淌，其中漂浮着一些深红色的星辰，有的很大，有的渺小，有的藏于深处，有的浮在表面。触碰星辰会将人拉入灰雾。被克莱恩拉入的人相当虚幻，染着微赤，就像深红星辰在灰雾之上的投影，这投影基于自己与深红的联系，切断这个联系，投影就会消散。

深红会隐藏在人的皮肤之下。
